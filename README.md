# Retro Galactic

Retro Galactic is a 2D endless space shooter developed with Unity.

The player controls a spaceship and fights waves of enemies while avoiding damage and collecting score.

## Gameplay Video
[![Watch the video](https://img.youtube.com/vi/4a9hm5AYlis/0.jpg)](https://www.youtube.com/watch?v=4a9hm5AYlis)

## Features
- Endless enemy waves
- Player shooting system
- Enemy pathfinding
- Health and damage system
- Score tracking
- Audio management
- Camera shake effects
- Wave configuration system

## Technologies
- Unity
- C#
